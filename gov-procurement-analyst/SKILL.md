## 详细功能说明

### 1. 商机发现与智能监控

**功能**：自动从各省市政府采购网、招投标平台采集公告，基于企业画像智能匹配。

**工作流**：
```
1. 读取用户订阅配置（行业/地区/金额/采购方式）
   - 如未配置 → 使用默认模板（全行业/全国/不限金额）
2. 调用 WebFetch 工具采集最新公告页面
   - 实时汇报进度："正在采集北京政府采购网... [3/10]"
3. 解析 HTML 提取关键字段
4. 对每个公告计算匹配分数（行业30%+资质25%+业绩20%+地区15%+金额10%）
5. 按匹配度降序输出，附带评分详情和 AI 建议
```

**匹配度评分**：

| 维度 | 权重 | 高匹配(≥80) | 中匹配(60-79) | 低匹配(<60) |
|------|------|-------------|---------------|-------------|
| 行业匹配 | 30% | 完全一致 | 相关/同类 | 跨行业 |
| 资质匹配 | 25% | 100%覆盖 | 覆盖≥60% | 覆盖<60% |
| 业绩匹配 | 20% | ≥3个同类 | 1-2个同类 | 无同类 |
| 地区便利 | 15% | 本地 | 同省 | 跨省 |
| 金额匹配 | 10% | ±50%范围内 | ±100%范围内 | 超范围 |

**分级标准**：
- 🔥 **高匹配（≥80分）**：优势明显，强烈建议投标
- ⭐ **中匹配（60-79分）**：值得考虑，注意补短板
- 📋 **一般（40-59分）**：备选，谨慎评估
- ❌ **低匹配（<40分）**：不建议投入

### 2. 投标决策分析

**功能**：给定采购项目，综合分析资质符合度、竞争对手历史、评分方法，输出"投/不投"明确建议。

**工作流**：
```
1. 接收项目链接或文件（PDF/Word/HTML）
2. 提取招标文件关键信息（评分方法/资质要求/预算/技术要求）
3. 资质符合性检查：企业资质 vs 招标要求
4. 竞争对手分析：同类型历史中标方统计
5. 报价策略计算：
   - 最低价法：历史中标价平均下浮×0.95 → 建议报价
   - 综合评分法：解析价格分公式 + 竞对模拟 → 最优区间
6. SWOT 分析 + 综合评分 + 风险等级 + 明确决策
```

**决策结论**：

| 结论 | 图标 | 含义 |
|------|------|------|
| 🔥 强烈建议 | score≥80 | 优势明显，全力投标 |
| ✅ 建议 | 60≤score<80 | 机会良好，正常参与 |
| ⚠️ 谨慎 | 50≤score<60 | 需补强，谨慎决策 |
| ❌ 不建议 | score<50 | 中标概率低，建议放弃 |

### 3. 投标文件智能编制

**功能**：上传招标文件后，自动解析评分标准、生成框架、撰写内容、输出自检清单。

**工作流**：
```
1. 接收 PDF/Word 招标文件
2. 提取全文内容（使用 Read 工具或 python-docx/pdfplumber）
3. 结构识别：投标须知/评分表/技术要求/合同条款
4. 生成评分自检清单（每项评分因素的达标策略）
5. 生成投标目录框架 + 逐章节内容
6. 输出格式自检报告（完整性/页码/盖章位置）
```

**输出包含**：
- 📋 评分自检清单（按评分表逐项列出准备要点）
- 📁 投标文件目录框架（标准格式）
- 📝 各章节撰写内容（用 LLM 生成技术方案 + 模板填充商务标）
- ✅ 格式自检报告

### 4. 供应商背调与合规审计

**功能**：一站式企业背调：诉讼/处罚/经营异常/关联关系/围标串标检测。

**数据源**：
- 国家企业信用信息公示系统（注册信息/股东/经营异常）
- 信用中国（行政处罚/失信惩戒/守信激励）
- 中国裁判文书网（诉讼记录/关联案件）
- 天眼查/企查查（关联企业图谱/最终受益人）

**背调报告结构**：
```
## 企业背调报告：{企业名称}

### 一、基础信息
- 注册信息/注册资本/成立日期/登记机关
- 经营范围/人员规模/企业规模

### 二、资质与认证
- 各类资质证书清单（含有效期）
- 资质到期预警

### 三、风险扫描
- ⚠️ 经营异常记录（列入/移出日期/原因）
- 🚫 行政处罚记录（处罚机关/处罚事由/处罚结果）
- 📜 重大诉讼（案由/标的额/审理阶段）
- 🔴 失信被执行人信息

### 四、关联分析
- 控股/参股企业清单
- 同一控制人/最终受益人关联企业
- 联系方式雷同检测

### 五、综合评级
- 信用等级：A（低风险）/ B（中风险）/ C（高风险）
- 建议：可合作 / 谨慎合作 / 不建议合作
```

### 5. 采购合规审计辅助

**功能**：自动检测采购流程合规风险点，适用于审计/纪检/内控部门。

**检测项目**：

| 检测项 | 法规依据 | 检查逻辑 |
|--------|----------|----------|
| 招标文件发售期 | 条例第31条 | 发售期 ≥5 日 |
| 投标截止时限 | 法律第35条 | 发出→截止 ≥20 日 |
| 澄清修改时限 | 条例第31条 | 澄清→截止 ≥15 日 |
| 中标公示期 | 条例第43条 | 公示期 ≥1 工作日 |
| 合同签订时限 | 法律第46条 | 通知书→签订 ≤30 日 |
| 评分异常 | 偏离均值>30% | 专家打分 Grubbs 检验 |
| 报价规律 | 等差/等比特征 | 多家报价序列分析 |
| 关联企业 | 同一控制/股东 | 关联图谱交叉比对 |
| 联系方式雷同 | 电话/邮箱/地址 | 不同投标人信息碰撞 |

### 6. 中标结果跟踪

**功能**：跟踪已投标项目的中标结果，统计分析竞对中标规律，复盘投标策略。

**跟踪指标**：
- 本次中标方是谁、中标金额多少
- 中标方与你的企业 SWOT 对比
- 中标价与预算的下浮比例（更新历史数据）
- 竞对累计中标次数与中标率更新
- 输在哪里/赢在哪里：复盘报告

### 9. 投标保证金管理

**功能**：计算投标保证金金额，检查是否合规，提醒缴纳和退还时间节点。

**计算规则**：
- 投标保证金 ≤ 项目预算的 2%
- 最高不超过 80 万元（工程招标）
- 政府采购货物/服务：通常 1%-2%

**检查项**：
- 保证金金额是否超比例
- 缴纳截止时间是否合理
- 退还时限是否合规（中标通知书发出后 5 日内退未中标人，合同签订后 5 日内退中标人）
- 是否接受保函形式

**输出模板**：
```
💰 投标保证金分析

项目预算：¥XXX 万
法定上限：¥XXX 万（2%）
招标文件要求：¥XXX 万
合规性：✅ 合规 / ❌ 超标

缴纳截止：XXXX-XX-XX
退还规则：
- 未中标人：中标通知书发出后 5 日内
- 中标人：合同签订后 5 日内

建议：...
```

### 10. 质疑投诉处理指引

**功能**：提供政府采购质疑投诉的完整流程指引和材料模板。

**流程**：
```
供应商认为采购文件/过程/结果损害权益
    │
    ▼
知道或应知道权益受损之日起 7 个工作日内
    │
    ▼
向采购人/代理机构提出质疑（书面）
    │
    ▼
采购人/代理机构 7 个工作日内答复
    │
    ├── 满意 → 流程结束
    │
    └── 不满意/未答复
        │
        ▼
答复期满后 15 个工作日内
        │
        ▼
向同级财政部门投诉
        │
        ▼
财政部门 30 个工作日内处理
        │
        ├── 驳回/维持 → 可行政复议或行政诉讼
        │
        └── 撤销/重新采购
```

**Skill 输出**：
- 质疑函/投诉书模板
- 证据材料清单
- 时效计算（精确到工作日）

### 11. 合同条款风险分析

- 违约责任（是否对等）
- 争议解决（管辖法院/仲裁）
- 知识产权归属
- 保密条款
- 不可抗力
- 合同变更/终止条件

**输出模板**：
```

━━━ 中风险条款 ━━━
...

━━━ 建议修改清单 ━━━
1. xxx
2. xxx

### 14. 电子卖场比价

**功能**：对比电子卖场（政府采购网上商城）中同类商品价格。

**数据源**：

### 15. 中小企业优惠政策计算

**功能**：计算中小企业参与政府采购的价格扣除优惠。

**政策依据**：
- 《政府采购促进中小企业发展管理办法》（财库〔2020〕46号）
- 小微企业价格扣除比例：10%-20%（由采购人设定）
- 联合体/分包扣除：4%-6%

**计算示例**：
```
项目预算：500 万
小微企业报价：480 万
价格扣除比例：15%
评审价：480 × (1-15%) = 408 万
最终以 408 万与其他供应商竞争

但合同仍按 480 万签订
```

**Skill 输出**：
- 中小企业声明函模板
- 价格扣除计算器
- 联合体协议模板
- 政策适用条件检查

### 快速体验工作流（用户未配置企业信息）

```

## 数据采集策略（重点合规要求）

### 数据采集原则

1. **仅采公开信息**：法律规定必须公开的采购公告/中标结果
2. **遵守 robots.txt**：每次请求前检查，拒绝爬取 Disallow 路径
3. **频率控制**：同一域名请求间隔 ≥3 秒，单域名 ≤50 次/小时
4. **明确标识**：User-Agent 使用 `GovProcurementBot/1.0 (WorkBuddy-CN)`
5. **缓存优先**：24 小时内不重复请求相同页面
6. **遇阻即停**：403/429/503 立即停止该域名采集

### 禁止行为（法律红线）

| 禁止行为 | 法律风险 |
|----------|----------|
| 破解验证码 | 违反《网络安全法》第27条 |
| 绕过登录/付费墙 | 违反《计算机信息系统安全保护条例》 |
| 使用代理池轮换 IP | 构成规避安全保护措施 |
| 伪造身份/指纹 | 构成身份欺诈 |
| 采集非公开信息 | 违反《个人信息保护法》 |
| 转售原始数据 | 违反平台使用条款 |

- 中国政府采购网（www.ccgp.gov.cn）

## 边界与限制

**此 Skill 不处理**：
- ❌ 政府采购预算编制（预算编制是采购人职责，但可辅助测算）
- ❌ 项目验收评审（需线下专家参与，但可提供验收标准分析）
- ❌ 政府采购投诉处理流程（涉及行政程序，但可提供流程指引和材料模板）
- ❌ 供应商财务审计（需注册会计师执行）
- ❌ 非政府采购的商业招标（如企业采购、军队采购需其他模块）

**数据限制**：
- 仅基于公开信息分析，无法访问内部/涉密数据
- 历史中标数据随平台更新可能有延迟（一般 1-3 天）
- 中标概率评估基于统计模型，不保证结果
- Skill 仅提供分析建议，最终决策由用户自行承担
