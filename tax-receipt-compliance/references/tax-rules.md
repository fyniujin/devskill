# 税务规则说明

本文档说明Skill涉及的税务规则和合规要求。

---

## 一、发票类型说明

### 1.1 增值税专用发票

| 项目 | 说明 |
|------|------|
| 定义 | 增值税一般纳税人销售货物或者提供应税劳务开具的发票 |
| 税率 | 13%、9%、6%、3%、1%、0% |
| 抵扣 | 可用于进项税额抵扣 |
| 代码位数 | 10位或12位 |
| 号码位数 | 8位 |

### 1.2 增值税普通发票

| 项目 | 说明 |
|------|------|
| 定义 | 增值税一般纳税人或小规模纳税人开具的普通发票 |
| 税率 | 13%、9%、6%、3%、1%、0% |
| 抵扣 | 不可用于进项税额抵扣（特殊情况除外） |
| 代码位数 | 10位或12位 |
| 号码位数 | 8位 |

### 1.3 增值税电子发票

| 项目 | 说明 |
|------|------|
| 定义 | 增值税发票的电子形式 |
| 税率 | 同纸质发票 |
| 抵扣 | 可用于进项税额抵扣 |
| 法律效力 | 与纸质发票具有同等法律效力 |
| 存储 | 需保存电子文件备查 |

## 三、发票管理规范

### 3.1 发票开具要求

根据《中华人民共和国发票管理办法》：

- 发票内容必须真实、准确
- 发票必须按照规定的时限、顺序、栏目开具
- 发票必须加盖发票专用章
- 不得虚开发票
- 不得转借、转让发票
- 不得代开发票

### 3.2 发票保管要求

- 发票存根联应当保存5年
- 发票登记簿应当保存5年
- 已开具的发票存根联和发票登记簿，保存期满后需报经税务机关查验后销毁
- 电子发票需保存电子文件

### 3.3 发票查验要求

- 企业应定期查验发票真伪
- 发现假发票应及时向税务机关报告
- 不得使用假发票入账
- 已抵扣的进项税额，如发票被认定为异常，应作进项税额转出

## 四、报销合规要求

### 4.1 报销凭证要求

| 要求项 | 说明 |
|--------|------|
| 发票真实 | 发票必须真实有效 |
| 内容完整 | 发票内容填写完整 |
| 金额准确 | 报销金额与发票金额一致 |
| 费用合理 | 费用发生与业务相关 |
| 时限合规 | 在规定期限内报销 |

### 4.2 常见不合规情况

- 使用假发票报销
- 虚增报销金额
- 报销个人费用
- 报销与业务无关的费用
- 超期报销
- 发票信息不完整

### 4.3 风险防范措施

- 建立发票查验制度
- 定期培训财务人员
- 建立举报机制
- 定期内部审计
- 使用技术手段辅助识别

---

## 五、税务风险提示

### 5.1 发票相关风险

| 风险类型 | 说明 | 后果 |
|----------|------|------|
| 假发票 | 使用伪造发票 | 补税、罚款、滞纳金 |
| 虚开发票 | 无真实交易开具发票 | 补税、罚款、刑事责任 |
| 代开发票 | 让他人代开发票 | 补税、罚款 |
| 发票丢失 | 发票保管不善 | 无法税前扣除 |

### 5.2 税务管理建议

1. **建立制度**：制定发票管理和报销管理制度
2. **培训人员**：定期培训财务人员
3. **技术辅助**：使用发票查验工具
4. **内部审计**：定期进行内部审计
5. **外部审计**：必要时聘请外部审计机构
