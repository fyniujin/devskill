# 采购框架协议风险审查清单

> 审查对象：采购框架协议（Framework Procurement Agreement）
> 覆盖维度：主体资格 | 价格条款 | 交付条件 | 验收标准 | 违约责任 | 特殊条款

---

## 一、主体资格审查（4项）

- [ ] **SC001** 采购方企业营业执照是否在有效期内，经营范围是否涵盖本项目
- [ ] **SC002** 供应商营业执照是否有效，是否具备所供货物/服务所需的行业资质
- [ ] **SC003** 签约代表是否持有法定代表人身份证明及授权委托书，授权范围是否包括签署本协议
- [ ] **SC004** 双方是否被列入经营异常名录或严重违法失信名单
- [ ] **SC005** 协议标的是否属于国家限制/禁止经营范畴，是否需要前置审批（如进出口许可）

## 二、价格条款审查（6项）

- [ ] **PR001** 价格确定方式是否明确（固定价格/浮动价格/按订单确认）
- [ ] **PR002** 固定价格是否约定有效期，到期后价格如何确定
- [ ] **PR003** 浮动价格是否设定合理的调价触发条件和调价公式
- [ ] **PR004** 价格是否含增值税、税率标准是否明确，税率变化时如何处理
- [ ] **PR005** 付款条件是否明确（付款节点、账期天数、付款前置条件）
- [ ] **PR006** 是否约定逾期付款的违约金或利息计算方式（建议不低于LPR的1.5倍）
- [ ] **PR007** 质保金比例是否合理（建议不超过合同总额5%），退还条件是否明确
- [ ] **PR008** 违约金与定金条款是否并存（民法典要求择一适用）

## 三、交付条件审查（6项）

- [ ] **DL001** 交付方式和具体地点是否明确（精确到门牌号或仓库名称）
- [ ] **DL002** 风险转移时点是否约定（建议交付验收后转移给需方）
- [ ] **DL003** 框架协议下订单提交方式、确认时限、交付周期是否明确
- [ ] **DL004** 分批交付的批次划分、每批数量及一批不合格对后续批次的影响
- [ ] **DL005** 包装标准、运输方式、保险安排及费用承担是否约定
- [ ] **DL006** 延迟交付的违约金和解除权是否约定（建议日0.2%，超15日可解除）

## 四、验收标准审查（5项）

- [ ] **AC001** 验收标准是否具体可量化（如引用国家标准GB/T XXXX）
- [ ] **AC002** 验收方法是否明确（全检/抽检比例/检测机构）
- [ ] **AC003** 验收期限是否合理（外观验收/性能验收分别约定）
- [ ] **AC004** 质量异议期限是否区分外观瑕疵和隐蔽瑕疵
- [ ] **AC005** 验收不合格的处理方式是否明确（修理/更换/退货/折价的选择权归属）
- [ ] **AC006** 是否约定第三方检测机构的选定方式和检测结果效力

## 五、违约责任审查（6项）

- [ ] **LB001** 双方违约责任是否对等（逾期付款与逾期交付的违约金比例应一致）
- [ ] **LB002** 违约金比例是否合理（一般违约日0.1%，根本违约总额20%以内）
- [ ] **LB003** 是否约定维权费用（律师费、保全费、鉴定费）由违约方承担
- [ ] **LB004** 协议解除权是否明确（解除条件、通知义务、解除后的结算方式）
- [ ] **LB005** 损失赔偿范围是否明确（是否包含可得利益损失）
- [ ] **LB006** 知识产权侵权保证条款是否包含（供方保证不侵犯第三方知识产权）
- [ ] **LB007** 保密义务是否明确（保密信息范围、保密期限、违反后果）

## 六、特殊条款审查（5项）

- [ ] **SP001** 不可抗力条款是否明确（定义范围、通知期限、后果处理）
- [ ] **SP002** 争议解决方式是否唯一明确（诉讼或仲裁二选一，不得同时约定）
- [ ] **SP003** 管辖地点是否有利（建议约定需方所在地）
- [ ] **SP004** 协议期限是否明确，自动续签条件是否合理（避免无限期锁定）
- [ ] **SP005** 反商业贿赂条款是否包含（承诺内容、违反后果、解除权）
- [ ] **SP006** 通知送达条款是否明确（送达地址、电子送达效力、视为送达条件）

---

> 使用说明：本清单共32项审查要点，覆盖采购框架协议核心风险维度。审查人员应逐项核查，对发现的问题标注风险等级（🔴高/🟡中/🟢低），并在审查结论中汇总说明。
