# 建设工程合同风险审查清单

## 一、主体资格审查

- [ ] 承包方是否具备与工程规模相适应的建筑业企业资质等级
- [ ] 承包方安全生产许可证是否在有效期内
- [ ] 承包方营业执照是否在有效期内，是否存在经营异常
- [ ] 承包方项目经理是否具备相应注册建造师资格
- [ ] 承包方技术负责人是否具备相应职称和执业资格
- [ ] 承包方近年是否有类似工程业绩
- [ ] 承包方财务状况是否良好，是否存在重大诉讼或仲裁
- [ ] 合同是否约定承包方不得转包工程
- [ ] 分包是否需经发包方书面同意，分包单位是否具备相应资质

## 二、工期条款审查

- [ ] 合同是否明确约定开工日期、竣工日期和总日历天数
- [ ] 是否约定开工通知程序和工期起算方式
- [ ] 是否约定工期顺延的情形（设计变更、工程量增加、不可抗力、发包方原因等）
- [ ] 是否约定工期顺延的申请程序和确认时限
- [ ] 逾期违约金计算方式是否合理（一般不超过合同总价的千分之一至千分之二每日）
- [ ] 逾期违约金累计上限是否合理（一般不超过总价3%-5%）
- [ ] 是否约定赶工费用的计算方法和确认程序
- [ ] 是否约定因承包方原因导致工期延误时发包方的解除权

## 三、质量标准审查

- [ ] 合同是否明确约定工程质量标准（合格/优良）
- [ ] 是否明确约定依据的国家标准和行业标准名称及版本
- [ ] 是否约定材料设备的质量标准和进场验收程序
- [ ] 是否约定不合格材料的退场费用承担
- [ ] 是否约定隐蔽工程覆盖前的验收通知时限（如48小时）
- [ ] 是否约定发包方逾期验收的处理方式
- [ ] 是否约定质量缺陷的修复时限和费用承担
- [ ] 是否约定二次验收的程序和费用承担
- [ ] 如约定工程质量需达到优良标准，是否约定创优奖励

## 四、付款节点审查

- [ ] 是否约定工程预付款的比例（一般10%-30%）
- [ ] 是否约定预付款的支付时间（开工前7日内）
- [ ] 是否约定预付款的扣回方式和时间
- [ ] 是否约定工程进度款支付节点（按月或按形象进度）
- [ ] 是否约定进度款支付比例（已完成工程价款的70%-80%）
- [ ] 是否约定进度款审核时限（如收到申请后14日内）
- [ ] 是否约定工程变更价款的确定方法
- [ ] 是否约定工程变更价款的调整时间
- [ ] 是否约定工程结算款支付期限（竣工验收合格后28日内支付至97%）
- [ ] 是否约定逾期支付工程款的利息（按同期贷款利率）
- [ ] 是否约定工程保留金比例（一般3%-5%）
- [ ] 是否约定保留金的返还时间和条件
- [ ] 是否约定发包方提供工程款支付担保

## 五、竣工验收审查

- [ ] 是否明确约定竣工验收的条件
- [ ] 是否明确约定竣工验收依据的标准
- [ ] 是否明确约定竣工验收的程序
- [ ] 是否约定竣工验收资料的清单、份数和交付时限
- [ ] 是否约定发包方擅自使用未验收工程的法律后果
- [ ] 是否约定验收不合格的处理方式和修复时限
- [ ] 是否约定二次验收的费用承担
- [ ] 是否约定甩项验收的条件和后续处理
- [ ] 是否约定竣工验收备案的责任主体和配合义务
- [ ] 是否约定竣工验收后工程移交的时间和程序

## 六、质保条款审查

- [ ] 质量保修期是否不低于法定最低期限
- [ ] 是否明确区分缺陷责任期和质量保修期
- [ ] 缺陷责任期是否合理（一般1年，最长2年）
- [ ] 是否约定保修期内承包方的维修时限（一般7日内，紧急情况24小时内）
- [ ] 是否约定保修期内维修费用的承担方式
- [ ] 是否约定质量保修金的返还条件和返还时间
- [ ] 是否约定质量保修金返还比例（不低于97%）
- [ ] 是否约定承包方不履行保修义务时发包方的救济措施

## 七、违约责任审查

- [ ] 是否约定发包方逾期付款的违约责任
- [ ] 是否约定发包方未提供施工条件的违约责任
- [ ] 是否约定发包方停工窝工损失的承担
- [ ] 是否约定承包方逾期竣工的违约责任
- [ ] 是否约定承包方工程质量不合格的违约责任
- [ ] 是否约定承包方擅自转包或违法分包的违约责任
- [ ] 是否约定单方解除合同的情形和程序
- [ ] 违约金的计算方式是否合理，是否过分高于实际损失

## 八、一般条款审查

- [ ] 是否约定不可抗力事件的范围
- [ ] 是否约定不可抗力发生后的通知义务
- [ ] 是否约定不可抗力损失的承担方式
- [ ] 争议解决方式（诉讼或仲裁）是否明确
- [ ] 管辖机构是否明确
- [ ] 协议适用法律是否明确
- [ ] 协议变更是否需双方书面同意
- [ ] 通知送达方式及地址是否明确
- [ ] 协议生效条件是否明确
- [ ] 协议完整性条款是否约定
