# 对赌协议风险审查清单

## 一、对赌方主体资格

- [ ] 对赌各方是否依法设立并有效存续，营业执照是否在有效期内
- [ ] 对赌方是否为国有企业，是否履行国资审批及评估备案程序
- [ ] 对赌方是否为外资企业，标的公司是否涉及外资准入限制或禁止类行业
- [ ] 对赌方是否存在被列入经营异常名录、严重违法失信名单等情形
- [ ] 自然人对赌方是否具备完全民事行为能力
- [ ] 对赌方是否具备签署对赌协议的合法授权（如董事会/股东会决议）
- [ ] 对赌方资金来源是否合法，是否存在代持或委托持股情形
- [ ] 对赌方与标的公司是否存在关联关系，是否涉及关联交易

## 二、标的公司主体资格

- [ ] 标的公司是否依法设立并有效存续
- [ ] 标的公司是否存在吊销、注销、清算等情形
- [ ] 标的公司股权是否存在质押、冻结或其他权利限制
- [ ] 标的公司是否存在未披露的关联交易或利益输送

## 三、业绩承诺条款

- [ ] 业绩承诺的指标是否明确（净利润、营收、EBITDA等）
- [ ] 业绩承诺指标的计算口径和会计准则是否明确
- [ ] 业绩承诺的审计机构是否明确约定
- [ ] 业绩承诺期限是否合理（通常不少于3年）
- [ ] 业绩承诺指标是否与历史业绩、行业平均水平匹配
- [ ] 是否约定业绩承诺的豁免情形（不可抗力、政策变化、重大疫情等）
- [ ] 业绩承诺条款是否具有可操作性和明确性

## 四、补偿机制条款

- [ ] 是否约定现金补偿条款
- [ ] 现金补偿的计算公式是否明确合理
- [ ] 是否约定股权补偿条款
- [ ] 股权补偿的调整比例是否明确
- [ ] 补偿义务人主体是否明确约定（控股股东或创始人）
- [ ] 补偿义务人不得为标的公司（参考《九民纪要》第5条）
- [ ] 补偿金额上限是否约定
- [ ] 补偿方式选择权归属是否明确（投资方或义务人）
- [ ] 补偿支付的时间节点是否明确
- [ ] 补偿义务人未按期支付的违约责任是否明确

## 五、股权回购条款

- [ ] 是否约定股权回购触发情形
- [ ] 股权回购触发情形是否明确列举（业绩不达标、IPO失败、创始人离职等）
- [ ] 股权回购的触发标准是否明确
- [ ] 回购价格及计算方式是否明确合理
- [ ] 回购义务人主体是否明确约定（控股股东或创始人）
- [ ] 回购义务人不得为标的公司（参考《九民纪要》第5条）
- [ ] 回购条款是否违反《公司法》第35条关于股东不得抽逃出资的规定
- [ ] 回购义务的履行期限是否明确
- [ ] 公司亏损时回购价格的调整机制是否约定
- [ ] 回购条款是否经过必要的内部审批程序

## 六、一票否决权条款

- [ ] 投资方是否享有对重大事项的一票否决权
- [ ] 一票否决权适用的重大事项清单是否明确列举
- [ ] 一票否决权的行使条件及程序是否明确
- [ ] 一票否决权是否与《公司法》第43条规定的特别决议2/3表决比例冲突
- [ ] 一票否决权的期限及终止条件是否明确
- [ ] 一票否决权条款的合法性和可执行性评估

## 七、估值条款

- [ ] 增资估值方法是否明确（收益法、市场法、资产基础法）
- [ ] 估值依据是否充分，是否有合格评估报告支持
- [ ] 增资价格与持股比例的计算是否合理
- [ ] 估值调整的触发条件是否明确（如业绩低于承诺的80%）
- [ ] 未分配利润及过渡期损益的归属是否明确
- [ ] 价格调整公式是否具有可操作性

## 八、上市对赌条款

- [ ] 是否约定IPO对赌条款（上市失败触发回购）
- [ ] IPO对赌条款的触发条件是否明确
- [ ] 上市对赌条款与主板/创业板/科创板上市规则的兼容性
- [ ] IPO对赌条款对公司上市审核的影响评估
- [ ] 上市失败的定义是否明确（如未在约定期限内申报/未成功上市）

## 九、信息披露与知情权

- [ ] 是否约定标的公司及原股东的定期信息披露义务
- [ ] 投资方查阅财务资料的权利是否充分（《公司法》第33条）
- [ ] 财务报告的提交频率及内容是否明确
- [ ] 投资方检查公司经营状况的权利及程序是否明确
- [ ] 重大事项的提前通知义务是否约定
- [ ] 投资方委派审计机构的权利是否明确

## 十、公司治理条款

- [ ] 对赌协议对标的公司治理结构的影响评估
- [ ] 董事会席位的分配及投资方委派董事的权利是否明确
- [ ] 董事会议事规则及表决方式是否明确
- [ ] 高级管理人员的任命权及提名程序是否明确
- [ ] 对赌协议与公司章程的衔接一致性
- [ ] 竞业限制条款是否约定（创始人及核心技术人员）

## 十一、违约责任

- [ ] 对赌各方违约责任的约定是否全面
- [ ] 违约情形是否明确（逾期出资、虚假陈述、违反竞业限制等）
- [ ] 违约金计算方式及比例是否合理
- [ ] 赔偿范围是否明确（直接损失、间接损失、律师费等）
- [ ] 合同解除条件及解除后的处理方式是否约定
- [ ] 对赌各方连带责任是否约定

## 十二、其他重要事项

- [ ] 对赌协议与标的公司章程是否存在冲突
- [ ] 对赌协议是否经过必要的内部审批程序（董事会/股东会）
- [ ] 争议解决方式（诉讼或仲裁）及管辖机构是否明确
- [ ] 合同生效条件是否明确
- [ ] 通知与送达条款是否完善
- [ ] 不可抗力条款是否约定
- [ ] 保密义务是否约定
- [ ] 合同完整性条款（entire agreement）是否约定
- [ ] 协议各方税费承担是否明确
- [ ] 对赌协议对外资准入和外汇管制的影响评估
