# 保密协议风险审查清单

## 一、主体资格审查

- [ ] 各方签约主体是否依法设立并有效存续
- [ ] 签约方营业执照是否在有效期内
- [ ] 签约方是否存在被列入经营异常名录、严重违法失信名单等情形
- [ ] 签约自然人是否具备完全民事行为能力
- [ ] 签约方是否为实际承担保密义务的主体
- [ ] 签约方是否具备签署协议的合法授权

## 二、保密范围审查

- [ ] 保密信息的定义是否明确、具体
- [ ] 是否以列举方式明确保密信息类型
- [ ] 保密信息是否具有商业秘密三要件（秘密性、价值性、保密性）
- [ ] 是否明确约定保密信息的标识方式（如标注"机密"或"保密"字样）
- [ ] 口头披露信息是否在约定期限内书面确认
- [ ] 保密范围是否涵盖衍生信息（分析、摘要、研究报告等）
- [ ] 是否排除已公开信息、公知信息和接收方已合法持有的信息
- [ ] 保密范围是否合理，是否不过于宽泛而影响可执行性

## 三、保密义务范围审查

- [ ] 接收方的保密义务范围是否明确
- [ ] 接收方可以向哪些关联方披露保密信息是否明确
- [ ] 关联方是否承担同等保密义务
- [ ] 关联方违反保密义务时接收方是否承担连带责任
- [ ] 接收方员工的保密义务是否约定
- [ ] 是否约定接收方应采取必要措施保护保密信息

## 四、保密期限审查

- [ ] 是否明确约定保密期限
- [ ] 保密期限是否合理，与信息的商业生命周期是否匹配
- [ ] 保密期限的起算时点是否明确
- [ ] 是否约定保密义务终止的情形
- [ ] 信息进入公共领域时保密义务是否自动终止
- [ ] 不同类型信息是否可约定不同保密期限
- [ ] 协议到期或终止后保密义务是否继续有效

## 五、违约责任审查

- [ ] 是否明确约定违反保密义务的违约责任
- [ ] 违约金金额或计算方式是否合理
- [ ] 违约金是否过高或过低
- [ ] 损失赔偿范围是否明确（直接损失、间接损失）
- [ ] 是否约定维权费用（律师费、公证费、诉讼费等）由违约方承担
- [ ] 是否约定继续履行责任（即使已违约仍应继续承担保密义务）
- [ ] 是否约定禁令救济条款
- [ ] 披露方有权无需证明实际损失即可申请行为保全

## 六、除外条款审查

- [ ] 是否约定保密义务的除外情形
- [ ] 除外情形是否包含：公开信息、接收方已合法持有、独立开发取得
- [ ] 是否约定法律强制披露时的处理程序
- [ ] 法律强制披露时是否需提前通知披露方
- [ ] 接收方主张除外情形时是否承担举证责任
- [ ] 除外条款是否包含独立开发或反向工程取得的类似信息

## 七、知识产权条款审查

- [ ] 是否明确约定保密信息相关的知识产权归披露方所有
- [ ] 披露行为是否不构成知识产权的转让或许可
- [ ] 接收方使用保密信息的范围是否受限
- [ ] 是否约定接收方不得基于保密信息申请专利或著作权

## 八、信息返还条款审查

- [ ] 是否约定协议终止或披露方要求时接收方应返还或销毁保密信息
- [ ] 信息返还或销毁的期限是否明确
- [ ] 是否约定接收方提供销毁证明
- [ ] 电子数据的删除是否约定
- [ ] 是否约定接收方留存的合规备份处理方式

## 九、一般条款审查

- [ ] 争议解决方式（诉讼或仲裁）是否明确
- [ ] 管辖机构是否明确（如披露方所在地法院）
- [ ] 协议适用法律是否明确
- [ ] 协议变更是否需双方书面同意
- [ ] 单方解除权的情形是否约定
- [ ] 通知送达方式及地址是否明确
- [ ] 保密义务是否不可转让
- [ ] 协议完整性条款（entire agreement）是否约定
- [ ] 协议生效条件是否明确

## 十、特殊情形审查

- [ ] 是否涉及跨境数据传输，是否需满足数据出境合规要求
- [ ] 是否涉及个人信息保护，是否需满足《个人信息保护法》要求
- [ ] 是否涉及国家秘密，是否需满足保密法要求
- [ ] 是否涉及竞业限制，是否需满足《劳动合同法》要求
- [ ] 是否涉及上市公司信息披露义务
- [ ] 是否涉及反垄断合规要求
