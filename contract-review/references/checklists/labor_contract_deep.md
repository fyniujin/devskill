# 劳动合同深度版风险审查清单

## 一、主体资格审查

- [ ] 用人单位是否依法设立并有效存续
- [ ] 用人单位营业执照是否在有效期内
- [ ] 用人单位是否存在被列入经营异常名录、严重违法失信名单等情形
- [ ] 劳动者是否年满16周岁（禁止招用童工）
- [ ] 劳动者是否已享受基本养老保险待遇（退休人员应签劳务协议）
- [ ] 签约主体与实际用工单位是否一致
- [ ] 是否存在假外包真派遣情形

## 二、劳动合同类型与期限审查

- [ ] 是否明确约定劳动合同类型（固定期限/无固定期限/以完成一定工作任务为期限）
- [ ] 是否存在连续工作满十年应订立无固定期限合同的情形
- [ ] 是否存在连续订立两次固定期限合同应订立无固定期限合同的情形
- [ ] 试用期期限是否符合法定上限
- [ ] 同一用人单位与同一劳动者是否只约定一次试用期
- [ ] 试用期工资是否不低于约定工资80%且不低于当地最低工资

## 三、劳动报酬审查

- [ ] 工资标准、构成是否明确约定
- [ ] 工资支付周期和支付日期是否明确
- [ ] 加班费计算基数是否约定且不低于法定标准
- [ ] 工资是否按月支付
- [ ] 是否存在拖欠或克扣工资的条款

## 四、竞业限制审查（重点）

- [ ] 被约定竞业限制的劳动者是否属于法定三类人员（高管/高技/保密义务人）
- [ ] 是否约定竞业限制经济补偿金
- [ ] 补偿金标准是否不低于前12个月平均工资的30%且不低于当地最低工资
- [ ] 补偿金支付时间是否为离职后按月支付（非在职期间随工资发放）
- [ ] 竞业限制期限是否超过二年法定上限
- [ ] 竞业限制地域范围是否合理
- [ ] 竞业限制业务范围是否明确具体
- [ ] 违约金金额是否合理（与补偿金比例是否失衡）
- [ ] 是否约定用人单位单方解除竞业限制的权利及额外补偿

## 五、培训服务期审查（重点）

- [ ] 培训是否属于专项技术培训（非岗前培训、入职培训）
- [ ] 是否约定服务期
- [ ] 违约金是否超过实际培训费用
- [ ] 违约金是否按服务期尚未履行部分分摊
- [ ] 培训费用凭证是否保留（培训费、差旅费、其他直接费用）
- [ ] 服务期超过劳动合同期限时是否约定自动续延

## 六、保密条款审查（重点）

- [ ] 是否约定劳动者保密义务
- [ ] 保密信息范围是否明确具体
- [ ] 保密期限是否约定
- [ ] 是否区分法定保密义务与约定竞业限制
- [ ] 违反保密义务的违约责任是否约定
- [ ] 保密义务是否延续至劳动合同终止后

## 七、劳动合同解除审查（重点）

- [ ] 用人单位单方解除条件是否限于法定情形（第39/40/41条）
- [ ] 是否违法增设解除条件
- [ ] 解除程序是否合法（通知工会、书面通知）
- [ ] 是否排除法定不得解除情形（孕期、产期、哺乳期、工伤、医疗期等）
- [ ] 违法解除赔偿金条款是否完整
- [ ] 经济补偿金计算基数和年限是否正确
- [ ] 是否存在"自动离职"、"自动放弃权利"等无效条款

## 八、劳动合同终止审查

- [ ] 终止条件是否限于法定情形（第44条）
- [ ] 是否存在"合同到期自动终止"等违法约定
- [ ] 是否违法增设终止条件
- [ ] 终止后的经济补偿是否约定

## 九、经济补偿与赔偿审查

- [ ] 经济补偿金支付情形是否完整
- [ ] 经济补偿金计算基数是否为解除前12个月平均工资
- [ ] 高收入员工经济补偿金是否受三倍社平工资和12年上限限制
- [ ] 违法解除赔偿金是否为经济补偿金的二倍
- [ ] 代通知金（N+1）情形是否约定

## 十、其他重要条款审查

- [ ] 工作地点约定是否明确
- [ ] 工作内容和工作岗位是否明确
- [ ] 工作时间和休息休假是否约定
- [ ] 社会保险和福利待遇是否约定
- [ ] 劳动保护、劳动条件和职业危害防护是否约定
- [ ] 竞业限制与保密条款是否独立于主合同可分割
- [ ] 争议解决方式是否明确
- [ ] 通知送达方式是否约定
