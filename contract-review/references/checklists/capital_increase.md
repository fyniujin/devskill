# 增资扩股协议风险审查清单

## 一、增资方主体资格

- [ ] 增资方是否依法设立并有效存续，营业执照是否在有效期内
- [ ] 增资方是否为国有企业，是否履行国资审批及评估备案程序
- [ ] 增资方是否为外资企业，标的公司是否涉及外资准入限制或禁止类行业
- [ ] 增资方是否存在被列入经营异常名录、严重违法失信名单等情形
- [ ] 增资方是否具备签署增资协议的合法授权（如董事会/股东会决议）
- [ ] 增资方资金来源是否合法，是否存在代持或委托持股情形
- [ ] 增资方与标的公司是否存在关联关系，是否涉及关联交易

## 二、标的公司主体资格

- [ ] 标的公司是否依法设立并有效存续
- [ ] 标的公司是否存在吊销、注销、清算等情形
- [ ] 标的公司股权是否存在质押、冻结或其他权利限制
- [ ] 标的公司其他股东是否放弃优先认购权
- [ ] 标的公司是否存在未披露的关联交易或利益输送

## 三、估值条款

- [ ] 增资估值方法是否明确（收益法、市场法、资产基础法）
- [ ] 估值依据是否充分，是否有合格评估报告支持
- [ ] 增资价格与持股比例的计算是否合理
- [ ] 是否约定业绩承诺及估值调整机制（对赌条款）
- [ ] 未分配利润的归属及处理方式是否明确
- [ ] 过渡期损益的归属是否明确
- [ ] 价格调整公式是否具有可操作性

## 四、反稀释条款

- [ ] 是否约定反稀释保护条款
- [ ] 反稀释调整方法是否明确（完全棘轮法或加权平均法）
- [ ] 反稀释例外情形是否约定（如员工期权、战略投资等）
- [ ] 是否约定优先认购权
- [ ] 反稀释条款的触发条件及计算公式是否明确
- [ ] 反稀释条款与后续融资的衔接是否明确

## 五、董事会席位

- [ ] 增资后董事会席位总数是否明确
- [ ] 增资方委派董事的人数及程序是否明确
- [ ] 董事会议事规则及表决方式是否明确
- [ ] 董事长、副董事长的委派权归属是否明确
- [ ] 董事会与股东会的职权划分是否明确
- [ ] 董事的任期、更换及辞职程序是否明确
- [ ] 董事会的通知程序及召开方式是否明确

## 六、一票否决权

- [ ] 增资方是否享有对重大事项的一票否决权
- [ ] 一票否决权适用的重大事项清单是否明确列举
- [ ] 一票否决权的行使条件及程序是否明确
- [ ] 一票否决权是否与公司法规定的特别决议表决比例冲突
- [ ] 一票否决权的期限及终止条件是否明确

## 七、股东会与表决机制

- [ ] 增资后股东会的表决机制是否明确
- [ ] 特别决议事项清单是否明确
- [ ] 特别决议的表决比例是否符合法定要求（2/3以上）
- [ ] 股东会的通知程序及召开方式是否明确
- [ ] 股东表决权一股一票还是特殊安排

## 八、增资价款支付

- [ ] 增资总价款及币种是否明确
- [ ] 增资价款的支付时间、方式是否明确
- [ ] 支付前提条件是否明确
- [ ] 增资方未按期足额缴纳的违约责任是否明确
- [ ] 出资方式是否合法（货币、实物、知识产权等）
- [ ] 非货币出资是否经过评估并办理财产权转移手续

## 九、监事会与高级管理人员

- [ ] 增资后监事会的组成是否明确
- [ ] 增资方委派监事的人数及程序是否明确
- [ ] 总经理、财务负责人等高级管理人员的任命权是否明确
- [ ] 高级管理人员的提名程序、任期及更换方式是否明确

## 十、退出机制

- [ ] 是否约定股权回购触发情形及回购价格
- [ ] 是否约定强制随售权（Drag-along）
- [ ] 是否约定优先清算权
- [ ] 是否约定共同出售权（Tag-along）
- [ ] 退出价格确定方式是否明确

## 十一、知情权与检查权

- [ ] 增资方查阅财务资料的权利是否明确
- [ ] 增资方检查公司经营状况的权利是否明确
- [ ] 财务报告的提交频率及内容是否明确
- [ ] 增资方委派审计机构的权利是否明确

## 十二、陈述与保证

- [ ] 标的公司及原股东的陈述与保证是否充分
- [ ] 陈述与保证的范围是否涵盖财务、税务、诉讼、合规等方面
- [ ] 陈述与保证不实的违约责任是否明确

## 十三、违约责任

- [ ] 违约情形是否明确（逾期出资、虚假陈述、违反竞业限制等）
- [ ] 违约金计算方式及比例是否合理
- [ ] 赔偿范围是否明确（直接损失、间接损失、律师费等）
- [ ] 合同解除条件及解除后的处理方式是否约定

## 十四、其他重要事项

- [ ] 争议解决方式（诉讼或仲裁）及管辖机构是否明确
- [ ] 合同生效条件是否明确
- [ ] 通知与送达条款是否完善
- [ ] 不可抗力条款是否约定
- [ ] 保密义务是否约定
- [ ] 合同完整性条款（entire agreement）是否约定
- [ ] 工商变更登记的办理义务及期限是否明确
- [ ] 竞业限制条款是否约定（创始人及核心技术人员）
- [ ] 协议各方税费承担是否明确
