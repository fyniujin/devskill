# 知识产权许可协议风险审查清单

## 一、主体资格审查

- [ ] 许可方是否为知识产权的合法权利人（核查专利证书/商标注册证/著作权登记证书）
- [ ] 许可方企业营业执照是否在有效期内
- [ ] 许可方是否存在被列入经营异常名录、严重违法失信名单等情形
- [ ] 知识产权是否为共有权利，如为共有是否取得全体共有人书面同意
- [ ] 被许可方主体资格是否合法有效
- [ ] 被许可方经营范围是否包含许可知识产权的使用领域
- [ ] 如许可方为知识产权的被授权人，核查其是否有权对外许可
- [ ] 知识产权是否存在质押、查封等权利限制情形

## 二、授权范围审查

- [ ] 是否明确约定许可性质（独占许可/排他许可/普通许可）
- [ ] 独占许可：排除第三方+许可方自身使用
- [ ] 排他许可：仅排除第三方，许可方自身可使用
- [ ] 普通许可：不排除任何人，许可方可继续许可第三方
- [ ] 是否明确列明许可的知识产权清单（专利号/商标注册号/作品登记号）
- [ ] 是否明确约定许可的具体权利范围（制造权/使用权/销售权/许诺销售权/进口权）
- [ ] 是否明确约定许可的使用方式（如生产特定产品/提供特定服务）
- [ ] 是否明确约定许可的产品范围或服务范围
- [ ] 授权范围是否与被许可方的实际需求相匹配
- [ ] 是否保留许可方对知识产权的控制权

## 三、地域限制审查

- [ ] 是否明确约定许可使用的地域范围（如中国大陆/全球/特定国家或地区）
- [ ] 地域限制是否合理，是否违反反垄断法规定
- [ ] 是否不当限制被许可方的出口权
- [ ] 许可地域是否在相应知识产权受法律保护的司法管辖区内
- [ ] 是否约定被许可方产品可出口的地域范围
- [ ] 是否约定违反地域限制的违约责任
- [ ] 地域限制是否构成市场分割或不合理竞争限制

## 四、转授权审查

- [ ] 是否明确约定被许可方是否有权向第三方转授权
- [ ] 如允许转授权，是否约定须经许可方书面同意
- [ ] 是否约定转授权的条件和程序
- [ ] 是否约定被许可方对次被许可方的行为承担连带责任
- [ ] 是否约定转授权费用的分配方式
- [ ] 是否约定转授权合同的期限不得超过原许可合同期限
- [ ] 是否约定转授权合同终止后次被许可方的义务
- [ ] 是否明确区分分许可与再许可的法律概念

## 五、改进成果审查

- [ ] 是否明确约定基于许可知识产权产生的改进成果的归属
- [ ] 改进成果归改进方/归许可方/双方共有是否明确
- [ ] 是否明确界定何为"改进"（衍生技术/变种技术/应用创新等）
- [ ] 是否约定改进成果的告知义务
- [ ] 是否约定改进成果的回授权（grant-back）许可
- [ ] 回授权的性质（独占/非独占）和范围是否明确
- [ ] 改进成果的使用限制是否明确
- [ ] 改进成果归属是否显失公平
- [ ] 是否约定改进方对改进成果的使用权利范围

## 六、许可费用审查

- [ ] 是否明确约定许可费用的计算方式（一次性/按销售额提成/按件计费/混合）
- [ ] 提成费的计算基数是否合理（净销售额/总销售额）
- [ ] 提成费计算基数是否约定扣除项目（税费/退货/折扣等）
- [ ] 是否约定最低许可费或最低使用量保证
- [ ] 许可费用的支付时间和支付方式是否明确
- [ ] 是否约定许可费用的审计权
- [ ] 是否约定逾期支付许可费用的违约责任
- [ ] 是否约定许可费用的调整机制

## 七、许可期限审查

- [ ] 许可期限是否明确约定
- [ ] 许可期限是否超过知识产权的法定保护期限
- [ ] 是否约定许可期限的起算时点
- [ ] 是否约定许可期限的续期条件
- [ ] 是否约定合同终止后的权利义务处理
- [ ] 是否约定库存销售宽限期
- [ ] 是否约定停止使用义务
- [ ] 是否约定保密义务继续有效

## 八、知识产权有效性审查

- [ ] 是否约定许可方维持知识产权有效性的义务
- [ ] 是否约定知识产权被无效后的许可费处理方式
- [ ] 是否约定知识产权被无效时被许可方的合同解除权
- [ ] 是否约定许可方应及时缴纳年费维持权利
- [ ] 是否约定许可方应对知识产权侵权行为采取维权措施
- [ ] 是否约定被许可方发现侵权时的通知义务

## 九、违约责任审查

- [ ] 是否明确约定违反许可协议的违约责任
- [ ] 违约金金额或计算方式是否合理
- [ ] 是否约定损失赔偿范围（直接损失/间接损失）
- [ ] 是否约定维权费用（律师费/公证费/诉讼费等）由违约方承担
- [ ] 是否约定继续履行责任
- [ ] 是否约定禁令救济条款
- [ ] 是否约定被许可方超范围使用的违约责任
- [ ] 是否约定许可方权利瑕疵的违约责任

## 十、合规性审查

- [ ] 许可条款是否违反反垄断法规定
- [ ] 是否包含搭售、限制转售价格等违法条款
- [ ] 专利/商标许可合同是否依法办理备案登记
- [ ] 涉外许可合同是否依法办理外汇登记、技术进口/出口登记
- [ ] 许可合同是否依法办理技术合同认定登记
- [ ] 是否涉及知识产权滥用行为
- [ ] 是否违反对外贸易法规定
- [ ] 是否涉及强制许可情形

## 十一、一般条款审查

- [ ] 争议解决方式（诉讼或仲裁）是否明确
- [ ] 管辖机构是否明确
- [ ] 协议适用法律是否明确
- [ ] 协议变更是否需双方书面同意
- [ ] 单方解除权的情形是否约定
- [ ] 通知送达方式及地址是否明确
- [ ] 保密义务是否约定
- [ ] 协议完整性条款是否约定
- [ ] 协议生效条件是否明确
- [ ] 合同附件是否齐全（知识产权清单/技术资料清单等）

## 十二、特殊情形审查

- [ ] 是否涉及标准必要专利许可（FRAND义务）
- [ ] 是否涉及开源软件许可
- [ ] 是否涉及跨境数据传输
- [ ] 是否涉及个人信息保护
- [ ] 是否涉及国家秘密
- [ ] 是否涉及上市公司信息披露义务
- [ ] 是否涉及反垄断合规要求
- [ ] 是否涉及出口管制合规要求
