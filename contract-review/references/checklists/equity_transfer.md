# 股权转让合同风险审查清单

## 一、转让方主体资格

- [ ] 转让方是否依法设立并有效存续，营业执照是否在有效期内
- [ ] 转让方是否为标的公司工商登记的合法股东
- [ ] 股权是否存在代持情形，实际出资人是否同意转让
- [ ] 转让方是否为国有企业，是否履行国资审批及评估备案程序
- [ ] 转让方是否为外资企业，标的公司是否涉及外资准入限制
- [ ] 转让方是否存在被列入经营异常名录、严重违法失信名单等情形
- [ ] 转让方是否具备签署合同的合法授权（如董事会/股东会决议）

## 二、股权比例与估值

- [ ] 转让股权比例是否明确，是否达到控股比例（67%、51%、34%等）
- [ ] 转让方是否已全面履行出资义务，是否存在出资瑕疵
- [ ] 标的公司股权是否存在质押、冻结或其他权利限制
- [ ] 转让价格是否经过评估，价格确定方法是否明确
- [ ] 标的公司是否存在未披露的关联交易
- [ ] 标的公司知识产权、核心资产权属是否清晰

## 三、转让价格与支付

- [ ] 转让总价款是否明确，币种是否约定
- [ ] 付款进度是否与股权交割、工商变更等关键节点挂钩
- [ ] 是否约定共管账户或托管支付安排
- [ ] 税务承担主体是否明确（所得税、印花税等）
- [ ] 是否约定付款前提条件（如审计报告、法律意见书等）

## 四、交割条件

- [ ] 是否明确约定交割条件清单
- [ ] 交割基准日是否明确，过渡期损益归属是否约定
- [ ] 是否约定交割前承诺与保证事项
- [ ] 是否约定交割后转让方的配合交接义务
- [ ] 交割条件未成就的处理方式是否明确

## 五、竞业限制

- [ ] 是否约定转让方的竞业限制义务
- [ ] 竞业限制的地域、期限、范围是否明确
- [ ] 是否约定竞业限制补偿金
- [ ] 是否约定核心技术人员的竞业限制义务
- [ ] 违反竞业限制的违约责任是否明确

## 六、优先购买权

- [ ] 是否已通知其他股东并确认放弃优先购买权
- [ ] 优先购买权的行使期限是否明确（不少于30日）
- [ ] 优先购买权的通知方式是否明确
- [ ] 同等条件的认定标准是否明确
- [ ] 其他股东放弃优先购买权的书面声明是否取得

## 七、股东会决议

- [ ] 是否取得标的公司同意转让的股东会决议
- [ ] 股东会决议的通知时间、表决方式是否符合章程及法定要求
- [ ] 决议通过的表决比例是否符合章程及法定要求
- [ ] 章程对股权转让是否有特殊规定

## 八、工商变更登记

- [ ] 是否明确约定工商变更登记的办理义务主体
- [ ] 工商变更登记的办理期限是否明确
- [ ] 未按期办理工商变更登记的违约责任是否约定
- [ ] 工商变更登记完成前的风险承担是否约定

## 九、债权债务承担

- [ ] 是否提供完整的资产负债表、债务清单
- [ ] 或有负债（担保、诉讼、税务风险等）是否充分披露
- [ ] 未披露债务的责任承担是否明确
- [ ] 转让方是否存在以公司资产为个人债务提供担保的情形
- [ ] 交割前后的债权债务归属是否明确

## 十、违约责任

- [ ] 违约情形是否明确（逾期付款、逾期交割、虚假陈述等）
- [ ] 违约金计算方式及比例是否合理
- [ ] 赔偿范围是否明确（直接损失、间接损失、律师费等）
- [ ] 合同解除条件及解除后的处理方式是否约定
- [ ] 争议解决方式（诉讼或仲裁）及管辖机构是否明确

## 十一、其他重要事项

- [ ] 合同生效条件是否明确
- [ ] 通知与送达条款是否完善
- [ ] 不可抗力条款是否约定
- [ ] 保密义务是否约定
- [ ] 合同完整性条款（entire agreement）是否约定
