# 合规检查清单

## 通用合规检查清单

### 合同主体
- [ ] 签约主体是否真实存在
- [ ] 签约主体是否有效存续
- [ ] 签约主体是否具有相应资质
- [ ] 签约代表是否有合法授权
- [ ] 签约主体是否涉及重大诉讼/仲裁

### 合同内容
- [ ] 合同标的是否明确
- [ ] 数量和质量标准是否明确
- [ ] 价款或报酬是否明确
- [ ] 履行期限、地点和方式是否明确
- [ ] 违约责任是否明确
- [ ] 争议解决方式是否明确

### 合同形式
- [ ] 合同是否采用书面形式
- [ ] 合同签署是否完整（签字/盖章）
- [ ] 合同日期是否填写
- [ ] 合同页码是否连续
- [ ] 合同附件是否齐全

### 法律合规
- [ ] 合同内容是否违反法律强制性规定
- [ ] 合同内容是否违反公序良俗
- [ ] 格式条款是否履行提示义务
- [ ] 权利义务是否对等

---

## 劳动合同专项检查清单

### 必备条款
- [ ] 用人单位名称、住所和法定代表人
- [ ] 劳动者姓名、住址和身份证号
- [ ] 劳动合同期限
- [ ] 工作内容和工作地点
- [ ] 工作时间和休息休假
- [ ] 劳动报酬
- [ ] 社会保险
- [ ] 劳动保护、劳动条件和职业危害防护

### 试用期
- [ ] 试用期时长是否符合法定上限
- [ ] 试用期工资是否不低于转正工资的 80%
- [ ] 试用期工资是否不低于当地最低工资标准
- [ ] 同一用人单位与同一劳动者是否只约定一次试用期

### 竞业限制
- [ ] 竞业限制人员是否限于高级管理人员、高级技术人员和其他负有保密义务的人员
- [ ] 竞业限制范围、地域是否合理
- [ ] 竞业限制期限是否不超过两年
- [ ] 是否约定按月支付经济补偿
- [ ] 经济补偿标准是否合理（不低于离职前12个月平均工资的30%）

### 解除合同
- [ ] 解除条件是否符合《劳动合同法》规定
- [ ] 经济补偿标准是否符合法定标准
- [ ] 是否违法约定违约金（除培训服务期和竞业限制外，不得约定违约金）

### 工资与福利
- [ ] 工资标准是否不低于当地最低工资标准
- [ ] 工资支付时间和方式是否明确
- [ ] 加班费计算基数是否明确
- [ ] 年休假是否约定
- [ ] 社保公积金是否依法缴纳

---

## 买卖合同专项检查清单

### 标的物
- [ ] 标的物名称、规格、型号是否明确
- [ ] 数量是否明确
- [ ] 质量标准是否明确（国标/行标/企标）
- [ ] 包装方式是否明确

### 交付与验收
- [ ] 交付时间、地点、方式是否明确
- [ ] 运输费用承担是否明确
- [ ] 验收标准和方法是否明确
- [ ] 验收期限是否合理
- [ ] 异议处理方式是否明确

### 价款与支付
- [ ] 合同总价是否明确
- [ ] 付款节点和比例是否合理
- [ ] 发票要求是否明确
- [ ] 质保金比例是否符合规定

### 风险与责任
- [ ] 风险转移时点是否明确
- [ ] 所有权保留条款是否约定
- [ ] 保修期和售后服务是否明确
- [ ] 违约责任是否对等

---

## 租赁合同专项检查清单

### 租赁物
- [ ] 租赁物名称、数量、用途是否明确
- [ ] 租赁物现状是否描述清楚
- [ ] 附属设施设备是否列明

### 租期与租金
- [ ] 租赁期限是否明确（不超过20年）
- [ ] 租金标准、支付方式是否明确
- [ ] 租金调整机制是否约定
- [ ] 押金金额和退还条件是否明确

### 使用与维护
- [ ] 租赁用途是否明确
- [ ] 维修责任是否明确
- [ ] 是否允许转租
- [ ] 装修和改善是否允许

### 解除与终止
- [ ] 提前解约条件是否公平
- [ ] 解约通知期限是否合理
- [ ] 合同终止后租赁物返还方式

---

## 技术开发合同专项检查清单

### 项目内容
- [ ] 项目名称、内容、范围和要求是否明确
- [ ] 技术指标和参数是否明确
- [ ] 研究开发计划是否明确

### 经费与支付
- [ ] 研究开发经费是否明确
- [ ] 经费支付方式是否合理
- [ ] 设备、器材、资料的财产权属是否明确

### 技术成果
- [ ] 知识产权归属是否明确
- [ ] 源代码是否交付
- [ ] 技术秘密保护是否充分
- [ ] 后续改进成果归属是否约定

### 验收与风险
- [ ] 验收标准和方法是否客观
- [ ] 风险责任承担是否合理
- [ ] 技术情报和资料的保密是否约定

---

## 融资协议专项检查清单

### 投资条款
- [ ] 投资金额和估值是否明确
- [ ] 股权比例是否明确
- [ ] 交割条件是否明确

### 特殊权利
- [ ] 对赌条款（业绩承诺、上市承诺）是否合理
- [ ] 反稀释条款是否合理
- [ ] 清算优先权是否合理
- [ ] 一票否决权范围是否合理
- [ ] 领售权/拖售权是否合理
- [ ] 回购条款是否合理

### 公司治理
- [ ] 董事会席位安排是否明确
- [ ] 知情权和检查权是否约定
- [ ] 跟投权是否约定

### 其他
- [ ] 陈述与保证是否充分
- [ ] 保密条款是否完善
- [ ] 违约责任是否明确
