# 企业微信语音消息 Agent

## ✨ 核心特性

- **零 API Key 依赖** - 企业微信内置 ASR 语音转文字免费使用
- **硬件自适应** - 自动检测用户电脑配置，调整并发和缓存策略
- **真正的天气查询** - v1.3.0 起接入 wttr.in 免费 API，中文描述 + 穿衣建议
- **当前时间查询** - 本地计算，100%可用，无需任何网络依赖
- **多轮对话** - 支持上下文保持的连续对话
- **性能优先** - 低配电脑也能流畅运行
- **安全可靠** - 所有数据处理在本地完成，不上传用户隐私
- **开箱即用** - 纯 Python 标准库，无需安装任何依赖

## 🚀 快速开始

### 一条命令体验所有功能

```bash
# 1. 一键测试（无需启动服务）
cd D:/skill/wecom-voice-agent
python scripts/wecom_webhook_server.py --quick

# 2. 单独测试某个指令
python scripts/voice_simulator.py --text "北京今天天气怎么样"
python scripts/voice_simulator.py --text "现在几点"
python scripts/voice_simulator.py --text "提醒我下午3点开会"
```

### 部署真实机器人

```bash
# 1. 启动回调服务器
python scripts/wecom_webhook_server.py --port 8080

# 2. 使用内网穿透暴露端口
ngrok http 8080
# 或
frpc http -l 8080 -p your-domain.com

# 3. 将穿透 URL 填入企业微信管理后台 → 智能机器人 → 回调 URL

# 4. 发送语音消息测试
```

> 📖 **详细部署指南**：参见 `references/step_by_step_setup.md`

## 🪤 避坑指南

| 常见坑 | 正确做法 |
|-------|---------|
| ❌ 在嘈杂环境发送语音指令 | ✅ 在安静环境说话，距离麦克风 20-30cm |
| ❌ 一次说多句话 | ✅ 一次只做一件事，分开发送 |
| ❌ 发送超过60秒的语音 | ✅ 控制在 60 秒以内 |
| ❌ 在群聊中发语音 | ✅ 只对机器人私聊发语音 |
| ❌ 发送方言 | ✅ 用普通话或粤语发送 |

## 📋 支持的语音命令

| 命令示例 | 功能 | 是否需要额外配置 |
|---------|------|---------------|
| "北京今天天气怎么样？" | ✅ 实时天气查询 | ❌ 不需要 |
| "现在几点？" | ✅ 当前时间查询 | ❌ 不需要 |
| "明天有什么会议？" | 🗓️ 日程查询 | ⚠️ 需管理员授权 |
| "提醒我下午3点提交报告" | 📝 创建待办 | ⚠️ 需管理员授权 |

| "你能做什么？" | 📖 获取帮助 | ❌ 不需要 |
| "退出语音模式" | 🚪 切换到文字模式 | ❌ 不需要 |

> 💡 **提示**：标注「⚠️ 需管理员授权」的功能，需要企业微信管理员在 work.weixin.qq.com 后台开启对应权限后才能真正执行。查询类功能（天气/时间）无需配置即可使用。

## ⚠️ 限制与边界

- 仅支持企业微信智能机器人场景
- 不支持主动外呼电话
- 语音消息限 60 秒以内
- 当前仅支持单聊（Phase 1）
- 方言支持有限（普通话/粤语/英文）

## 🔐 隐私声明

- 本技能不存储、不上传、不转发用户的任何语音数据
- 语音转写完全由企业微信官方接口完成
- 所有文本数据仅保存在本地会话缓存中
- 会话过期后自动清理
